## 📌 About  

The **iO Beats Player** is the official **Web + Mobile bridge** of the IO Beats ecosystem.  
It connects streaming, NFTs, DeFi utilities, and wallet interactions — all within a unified interface.  

🎵 Designed for artists, listeners, and investors alike:  
- Stream & discover Web3 music  
- Claim NFT albums and earn royalties  
- Connect wallets across multiple chains  
- Bridge DeFi and music in one click  

> “From the first beat to the blockchain — **Own the Music, Earn the Future.**”

---

## 🧠 Core Features  

| Category | Description |
|-----------|--------------|
| 🎶 **Streaming Engine** | High-quality MP3 / HLS adaptive audio with progress tracking |
| 🖼 **NFT Integration** | ERC-721 & ERC-1155 music albums with royalties (EIP-2981) |
| 💰 **DeFi Utilities** | On-chain staking, vaults, and listener rewards |
| 🔗 **Wallet Connect** | Secure login via WalletConnect, RainbowKit, or Web3Modal |
| 📱 **Mobile Bridge** | Deep link routing, WebView sync, and native event tracking |
| 📊 **Analytics** | Transparent user & transaction metrics for DAO reports |

## 🧩 Tech Stack  

- **Frontend:** React + TypeScript + Tailwind  
- **Backend:** Node.js + Express + IPFS pinning  
- **Web3:** Ethers.js + Wagmi + WalletConnect  
- **Streaming:** HLS.js + custom audio service worker  
- **State Management:** Zustand / Context API
